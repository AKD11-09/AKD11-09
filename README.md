## 🧠 About Me

I'm a software developer focused on applying **deep learning to geospatial problems** — training and deploying models that extract meaning from spatial and remote sensing data in ways traditional GIS tooling simply can't.

My work spans the full pipeline: from raw geographic data ingestion and preprocessing, through model design and training, to serving predictions via scalable APIs backed by spatial databases.

- 🛰️ Designing deep learning models for domain-specific geospatial tasks — segmentation, classification, detection, and prediction on spatial data
- 🗺️ Bridging the gap between classical GIS workflows and modern ML infrastructure
- 🐍 Python-first development across the entire stack — data, model, and backend
- ☁️ Deploying production-grade geospatial services on cloud infrastructure
